# CHUNG COMPANY

> AI가 운영하는 글로벌 유튜브 콘텐츠 제작 회사

---

## 프로젝트 소개

CHUNG COMPANY는 AI를 이용하여 여러 개의 유튜브 채널을 자동으로 운영하는 시스템을 개발합니다.

이 프로젝트의 목표는 단순한 영상 생성이 아니라,

**AI가 유튜브 채널을 기획하고 성장시키는 것**입니다.

---

## 핵심 목표

사용자는

- 어떤 채널을 만들지 결정한다.
- 어떤 주제로 영상을 만들지 결정한다.

나머지는 AI가 수행한다.

- 리서치
- 대본 작성
- 장면 구성
- 이미지 생성
- 영상 생성
- 음성 생성
- 자막 생성
- 품질 검사
- 자동 수정
- 업로드 준비
